# Introduction

LLM-based AI coding tools are completely changing the way many software engineers operate today.
If you think AI is completely transforming software engineering today and want to ride this wave,
this course is for you.  If you are an AI skeptic, I will make no attempt to convince you otherwise.

Many of today's software companies are not looking for junior engineers because AI models can now write code at
the quality level of mid-level engineers.  If you want to get hired, you need higher level systems
experience, the type of experience that you generally only obtain while working at other companies.
But if you can only become a senior engineer by working, but nobody wants to hire you, now what?
Bridging that gap is the purpose of this course.

The goal of this course is to help you develop the intuition of a senior level backend engineer in
a web programming context, which is my area of expertise.  To benefit the most, you are expected to
use the AI tools of your own choice, but just know that I use Claude Code.  In my experience, even
a $20/month subscription is sufficient to gain tremendous value.

In this course, I assume that you are a novice that knows next to nothing about professional programming,
so I'm going to teach you from the ground up, so even a middle school student should be able to learn
from this course.

I am the type of person that learns by example, so rather than only teach you theory, each lesson will come
with at least one assignment, and I'll even provide sample solutions.  In many cases, I will even provide
quizzes to help you solidify your understanding.  In fact, I believe the quizzes will be instrumental
in accelerating your ability to learn the curriculum.

I also firmly believe that if you want to be a senior engineer, you need to navigate Linux like a pro, with
the bonus that if you are using a Mac, the terminal navigates just like Linux.  Therefore, the early modules
teach you Linux basics in a structured way.  If you are already a pro at Linux/Mac, you can naturally skip
those modules.

# Recommended OS

Although you could make this work with a Windows machine, I highly recommend a Mac or Linux environment.
This course will teach you how to think like a backend engineer, and like it or not, most webservers are
Linux-based, so in the circles where I run, it's safe to say that nearly everybody uses a Mac laptop, which
internally operates like Linux.  If you use Windows, I don't know which lessons will prove difficult to use,
but I hope you can follow along!
