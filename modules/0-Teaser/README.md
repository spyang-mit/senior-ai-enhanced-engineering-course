# Module 0 - Teaser

The purpose of this module is to immediately open your eyes to how quickly AI can generate a
simple program in a matter of minutes that would have in the past taken a good engineer many
hours to build.

You can do this exercise with any AI programming agent, but here's some specific guidance for how
to do it with Claude Code:

- Step 0: Purchase Claude Code for $20/month (or more if you wish)
- Step 1: install the Claude CLI (look up directions yourself on the web)
- Step 2: open up a terminal window, cd to the directory where you want to work, and type "claude" to start the CLI
- Step 3: Enter your prompt and see what happens.

## Assignment

Ask Claude to create a single page web application using basic HTML.  It's overkill to use something fancy like
React, so you can specify vanilla Javascript or perhaps JQuery for simplicity.  You can do whatever you want,
but I have a specific assignment you can try, accompanied by a sample solution in this folder.  Here it is:

Create a Pokemon type matchup quiz.  If you don't know about Pokemon type matchups, you can even ask AI to explain it to you.

Programming is an iterative process, so after the first output, deliver feedback to AI and it will revise
the program to your specifications.

## Hints

Quite frankly, if you literally just prompt "Create an HTML file that quizzes you on Pokemon type matchups", AI
will likelly generate something reasonable.  However, if you know more about type matchups and want to influence the
design, you can make your prompt more specific.  Really try to do this yourself first, then when you are done,
you can view the sample-solution folder to see what I personally did.
